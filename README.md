<h1>EagleAdmin1.0.0</h1>

基于<a href="https://www.workerman.net/doc/webman/" target="_blank">webman</a>(高性能HTTP服务框架)开箱即用的高质量中后台管理系统

<h1>安装</h1>

composer环境的安装命令如下

``` bash
composer require huguijian/eagleadmin
```

如果无法直接安装，采用下面的安装命令

``` bash
composer require -W huguijian/eagleadmin
